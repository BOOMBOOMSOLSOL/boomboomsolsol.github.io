개인정보처리방침

시행일: 2025년 3월 22일 · 붐붐솔솔

1. 수집하는 개인정보

붐붐솔솔(이하 "회사")이 운영하는 앱 서비스(이하 "서비스")는 아래 정보를 수집합니다.

필수 수집 항목

  • 계정 정보: 소셜 로그인(Google, Apple) 시 이메일 주소
  • 별명: 사용자가 직접 입력한 닉네임 (최대 6글자)
  • 학습 데이터: 학년 선택, 학습 진행도, 퀴즈 응답 기록, XP, 스트릭

자동 수집 항목

  • 기기 정보: 운영체제 종류 및 버전 (앱 안정성 개선 목적)
  • 앱 사용 기록: 오류 로그, 앱 실행 및 사용 빈도 (서비스 개선 목적)

수집하지 않는 항목

위치 정보, 연락처, 사진, 통화 기록, 광고 식별자 등 민감한 개인정보는 수집하지 않습니다.

2. 개인정보의 이용 목적

  • 사용자 식별 및 로그인 기능 제공
  • 학습 진행도 저장 및 기기 간 동기화
  • 서비스 개선, 오류 수정 및 신규 기능 개발
  • 이용자 문의 응대 및 공지사항 전달

3. 개인정보의 보관 및 파기

  • 학습 데이터는 사용자 기기(로컬 DB)에 우선 저장되며, 로그인 시 서버에 동기화됩니다.
  • 서버에 저장된 데이터는 계정 삭제 요청 시 30일 이내 파기합니다.
  • 회원 탈퇴 시 모든 개인정보를 지체 없이 파기합니다.
  • 관련 법령에 의해 보존이 필요한 경우, 해당 기간 동안 별도 보관 후 파기합니다.

4. 개인정보의 제3자 제공

회사는 사용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 아래의 경우는 예외로 합니다.

  • 법령에 의해 요구되는 경우
  • 사용자가 사전에 동의한 경우

5. 개인정보의 위탁 및 국외 이전

회사는 서비스 제공을 위해 아래 업체에 개인정보 처리를 위탁하고 있으며, 일부 데이터가 국외로 이전됩니다.

수탁업체위탁 업무보관 국가이전 항목
Supabase Inc.데이터 저장 및 인증미국이메일, 학습 데이터
Google LLC소셜 로그인, 앱 배포미국이메일, 기기 정보

각 수탁업체는 자체 개인정보보호 정책에 따라 데이터를 안전하게 관리합니다.

6. 이용자의 권리

사용자는 언제든지 아래 권리를 행사할 수 있습니다.

  • 데이터 열람: 앱 내 프로필/통계 화면에서 본인의 학습 데이터 확인
  • 데이터 정정: 별명 등 개인정보의 수정
  • 계정 삭제: 앱 내 설정 > 계정 관리에서 계정 삭제 가능
  • 동의 철회: 앱 내 설정에서 로그아웃 또는 계정 삭제로 동의 철회 가능

7. 비로그인(게스트) 사용

로그인 없이도 앱을 사용할 수 있습니다. 이 경우 모든 데이터는 기기 내에만 저장되며, 서버로 전송되지 않습니다. 앱 삭제 시 데이터도 함께 삭제됩니다.

8. 아동의 개인정보 보호

회사는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 이용자가 개인정보를 제공한 사실을 알게 된 경우, 해당 정보를 즉시 삭제합니다. 법정대리인이 아동의 개인정보 열람, 정정, 삭제를 요청할 경우 지체 없이 처리합니다.

9. 개인정보의 안전성 확보 조치

  • 데이터 전송 시 SSL/TLS 암호화 적용
  • 비밀번호 등 중요 정보의 암호화 저장
  • 접근 권한 최소화 및 관리

10. 개인정보처리방침의 변경

본 방침이 변경되는 경우, 앱 내 공지 또는 본 웹사이트를 통해 변경 사항을 안내하며, 변경된 방침은 공지한 날로부터 7일 후 시행됩니다.
