1. 수집하는 개인정보

국어:어휘의 신(이하 "앱")은 서비스 제공을 위해 아래 정보를 수집합니다.

필수 수집 항목

  • 계정 정보: 소셜 로그인(Google, Apple) 시 이메일 주소
  • 별명: 사용자가 직접 입력한 닉네임 (최대 6글자)
  • 학습 데이터: 학년 선택, 학습 진행도, 퀴즈 응답 기록, XP, 스트릭

자동 수집 항목

  • 기기 정보: 운영체제 종류 및 버전 (앱 안정성 개선 목적)

수집하지 않는 항목

위치 정보, 연락처, 사진, 통화 기록 등 민감한 개인정보는 수집하지 않습니다.

2. 개인정보의 이용 목적

  • 사용자 식별 및 로그인 기능 제공
  • 학습 진행도 저장 및 기기 간 동기화
  • 서비스 개선 및 오류 수정

3. 개인정보의 보관 및 파기

  • 학습 데이터는 사용자 기기(로컬 DB)에 우선 저장되며, 로그인 시 Supabase 서버에 동기화됩니다.
  • 서버에 저장된 데이터는 계정 삭제 요청 시 즉시 파기합니다.
  • 회원 탈퇴 시 모든 개인정보를 지체 없이 파기합니다.

4. 개인정보의 제3자 제공

앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 단, 아래의 경우는 예외로 합니다.

  • 법령에 의해 요구되는 경우
  • 사용자가 사전에 동의한 경우

6. 이용자의 권리

  • 데이터 열람: 앱 내 프로필/통계 화면에서 본인의 학습 데이터 확인
  • 계정 삭제: 앱 내 설정 > 계정 관리에서 계정 삭제 가능
  • 로그아웃: 앱 내 설정에서 로그아웃 가능

7. 비로그인(게스트) 사용

로그인 없이도 앱을 사용할 수 있습니다. 이 경우 모든 데이터는 기기 내에만 저장되며, 서버로 전송되지 않습니다.
